# 🔐 RyzeAuth

An advanced Authentication System for Minecraft server (Spigot/Paper)

> **Note:** This plugin is currently in beta.  
> For suggestions or bug reports, contact me on Telegram: [@xoxRyze](https://t.me/xoxRyze)

## 🧾 Commands

* `/register <password> <password>`
* `/login <password>`
* `/changepassword <password> <new password>`
* `/unregister <password> `
* `/unlogin <password> `
 
* `/adminauth changepassword <player> <password>`
* `/adminauth unregister <player>`
* `/adminauth register <player> <password>`
* `/adminauth kick <player> <reason>`
* `/adminauth checkip <player>`
* `/adminauth dupeip <player>`
* `/adminauth reload`
 
## 🔐 Permissions

* `/unlogin` » `ryzeauth.command.unlogin`
* `/unregister` » `ryzeauth.command.unregister`
* `/changepassword` » `ryzeauth.command.changepassword`
* `/adminauth` » `ryzeauth.command.adminauth`
* `/adminauth dupeip` » `ryzeauth.command.adminauth.dupeip`
* `/adminauth checkip` » `ryzeauth.command.adminauth.checkip`
* `/adminauth reload` » `ryzeauth.command.adminnauth.reload`
* `/adminauth kick` » `ryzeauth.command.adminauth.kick`
* `/adminauth unregister` » `ryzeauth.command.adminauth.unregister`
* `/adminauth changepassword` » `ryzeauth.command.adminauth.changepassword`
* `/adminauth register` » `ryzeauth.command.adminauth.register`
